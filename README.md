# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/mananpatel17106-web/Streamix-Frontend.git
```

---

## Frontend

```bash
cd Streamix-Frontend
npm install
npm run dev
```

---

# 🔐 Environment Variables

Create `.env`

```env
VITE_API_BASE_URL=
```

---

# 📦 Build

Frontend

```bash
npm run build
```

Backend

```bash
npm start
```
